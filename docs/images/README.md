## How to Capture Screenshots

### Option 1: Chrome DevTools (Recommended)

1. Open the page in Chrome
2. Press `Cmd+Option+I` (Mac) or `Ctrl+Shift+I` (Windows) to open DevTools
3. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows) to open Command Palette
4. Type "screenshot" and select:
   - **Capture screenshot** - captures visible area
   - **Capture full size screenshot** - captures entire page
5. Save the file to this folder with the correct name

### Option 2: macOS Screenshot

1. Navigate to the page
2. Press `Cmd+Shift+4` then `Space` to capture the window
3. Click on the Chrome window
4. Rename and move the file to this folder

### Recommended Viewport Size

For consistent screenshots, set your browser window to approximately:
- Width: 1400px
- Height: 900px
